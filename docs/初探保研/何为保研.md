# 何为保研

**保研** = **获得校内名额**(9月底依据政策评审获得) + **获得目标院校院系的录取资格**(参与夏令营，预推免， 九推)

## 总体流程

![062f6fd600b56024b8748d6329d889c5](何为保研.assets/062f6fd600b56024b8748d6329d889c5.png)

如图可见，在大三学期的寒假之前，是大家的正常学习，竞赛，科研。在大三的寒假以及之后，则要开始准备保研事宜。
